# User Needs Document
As a user, I want a way to automatically create a PDF report for my threat modeling activites.
As a user, I want to keep on using my current method of threat modeling, which outputs a CSV file with threats, their description, statuses, and justifications for the statuses among other columns.
As a user, I want the reports to be created using Typst so that I can easily maintain them through time.
As a user, I want the way to automatically create the PDF report to be useable on Windows or Linux.

As an IT admin, I want the way to automatically create the PDF report to use safe interaction techniques so that detection systems are not falsely alerted by its use.
As an IT admin, I want the user to be able to accelerate their work without compromising their security of the security of the company.

As a coordinator, I want the user to accelerate their workflow.
As a coordinator, I want the user to be replaceable in case of resignation, so the tool should be very well documented and easy to pick up.
